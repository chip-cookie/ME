# Content Directory

This directory contains MDX content files for the blog/documentation.

## Structure
- `posts/` - Blog posts in MDX format

## Frontmatter Fields
Each MDX file should include:
- `title` (required): Post title
- `slug` (required): URL slug
- `description`: Short description
- `author`: Author name
- `date`: Publication date (YYYY-MM-DD)
- `tags`: Array of tags
- `featured`: Boolean for featured posts
