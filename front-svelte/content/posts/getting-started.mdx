# JasoS에 오신 것을 환영합니다! 🎉

JasoS는 AI를 활용하여 채용 공고를 분석하고, 맞춤형 자기소개서를 작성하는 플랫폼입니다.

## 주요 기능

### 1. 📊 채용 공고 분석
채용 공고를 업로드하면 AI가 핵심 키워드와 기업 정보를 자동으로 분석합니다.

### 2. ✍️ 스타일 학습
기존에 작성한 합격 자소서를 업로드하면 AI가 여러분의 문체를 학습합니다.

### 3. 💬 면접 준비
분석된 정보를 바탕으로 예상 면접 질문을 생성하고 연습할 수 있습니다.

## 시작하기

1. **스타일 학습** 메뉴에서 기존 자소서를 업로드하세요.
2. **자소서 작성** 메뉴에서 채용 공고를 분석하세요.
3. AI가 생성한 초안을 검토하고 수정하세요.

> 💡 **팁**: 여러 개의 합격 자소서를 업로드할수록 AI가 더 정확하게 문체를 학습합니다!
